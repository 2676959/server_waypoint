## Community Translators

Thank you to the community translators who help make Server Waypoint available to more players in more languages.

Translation contributors who would like to be credited can add themselves to [`CREDITS.txt`](./CREDITS.txt) in a pull request. If you do not have a GitHub account, you can contact me on Discord instead.

Credit format: `Preferred Name` followed by an optional link.

Credit pull request guidelines:
- Edit only [`CREDITS.txt`](./CREDITS.txt) unless a translation file also needs to be added or updated.
- Add one credit per line under the `Translation contributors` section using the credit format above.
- Include the language or locale you contributed to in the pull request description.
- Use only a name, handle, or link that belongs to you and that you are comfortable publishing.
- Keep links relevant to your identity or translation work, such as a GitHub profile, website, or translation platform profile.
